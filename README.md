# 这b学上的值不值 · 测算版

> 大学主要矛盾：高度理论化的专业课程 × 几乎为零的就业指导

量化你的大学就业 ROI，通过拔河模型告诉你：**该留在学校还是跑出去实习？**

🔗 在线体验：https://bxue-value.vercel.app/

---

## 玩法

填写 15 个维度的选择，实时看到「在校 vs 实习」拔河条变化，点击提交后生成完整的大学诊断报告。

### 评估维度

| 模块 | 维度 |
|------|------|
| **硬件基盘** | 学校层级、专业类别、所在城市等级、校区区位 |
| **校内资源** | 老师产业连接度、校招直通车质量、校友网络、课程辅导、师生比、校内活动、GTA/RA 岗位 |
| **行政灵活度** | 考勤严格度、学分置换、请假难度、GPA 评定难度 |

### 结果等级

| 等级 | 标签 | 建议 |
|------|------|------|
| T0 | 天选象牙塔 | 享受大学，校内拔尖 |
| T1 | 回血型避风港 | 内外兼修，重心校内 |
| T2 | 薛定谔的大学 | 五五开，必须找实习 |
| T3 | 自我救赎训练营 | 翘课实习，保学位 |
| T4 | 纯纯就业绊脚石 | 快跑，学校当宿舍 |

## 计算公式

```
majorBlend = 0.45 + major × 0.4
score = (brand × majorBlend + resource × 0.35 + admin × 0.1) × (1 - pullDiscount)
```

- **brand**：学校品牌分（清北=10，普通双非=2）
- **majorBlend**：压缩专业系数（CS=0.85，文科=0.51），热门专业学校和实习都好，不给满额品牌加成
- **resource**：7 个校内资源子维度的平均分
- **admin**：4 个行政灵活度子维度的平均分
- **pullDiscount**：实习拉力折扣，`geoAdvantage / 10 × 0.35`，城市好+区位好=实习机会多=学校边际价值下降

分数范围 0-10，越高越值得留在学校，越低越应该出去实习。

详细公式说明见 [scoring.md](scoring.md)。

## 技术栈
