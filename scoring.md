# b学上的值不值 · 评分标准 v2

## 核心逻辑

最终分数 = **学校对你就业的帮助程度**（0-10分）

- 分越高 → 学校值得留（在校 > 实习）
- 分越低 → 实习更值得（实习 > 在校）

---

## 公式

```
majorBlend = 0.45 + major × 0.4
score = (brand × majorBlend + resourceAvg × 0.35 + adminAvg × 0.1) × (1 - pullDiscount)
```

### 1. 学校品牌 brand（0-10）

| 学校层级 | 分数 |
|---------|------|
| 清北/华五 | 10 |
| 中等985 | 8.5 |
| 末流985/强势211 | 7 |
| 普通211 | 5.5 |
| 双一流(非211) | 4 |
| 强势双非(深大/杭电) | 3.5 |
| 普通双非 | 2 |
| 民办/专科 | 0.5 |

### 2. 专业系数 majorBlend（0.3~0.85）

**压缩后的专业系数**：热门专业学校和实习都好，不该给满额品牌加成；冷门专业学校帮助有限，但也不该打到骨折。

`majorBlend = 0.45 + major × 0.4`

| 专业类别 | raw major | majorBlend | 逻辑 |
|---------|-----------|------------|------|
| 计算机/金融/量化 | 1.0 | **0.85** | 学校好，但实习同样好，品牌不给满额 |
| 电子/通信/自动化 | 0.8 | **0.77** | 同上，略弱 |
| 传统工科 | 0.5 | **0.65** | 学校帮助中等 |
| 理学 | 0.5 | **0.65** | 同上 |
| 管理/经济/法学 | 0.35 | **0.59** | 学校帮助偏弱 |
| 文科/社科/艺术 | 0.15 | **0.51** | 学校帮助很弱，但不为零 |

### 3. 校内资源 resourceAvg（0-10）

7个子维度等权平均：

| 维度 | 10分 | 5分 | 0分 |
|------|------|-----|-----|
| 老师产业连接度 | 直接内推实习 | 偶尔推荐 | 纯学术 |
| 校招直通车质量 | 大厂免笔试 | 有宣讲会 | 走过场 |
| 校友网络活跃度 | 天天发内推 | 偶尔有 | 死群 |
| 课程辅导资源 | 免费辅导+mentor | 有但一般 | 没有 |
| 师生比 | <15:1 | ~25:1 | >50:1 |
| 校内活动知名度 | 企业赞助竞赛 | 偶尔有 | 几乎没有 |
| GTA/RA竞争度 | 轻松拿到 | 需要竞争 | 卷也没用 |

### 4. 行政灵活度 adminAvg（0-10）

4个子维度等权平均：

| 维度 | 10分 | 5分 | 0分 |
|------|------|-----|-----|
| 考勤严格度 | 随便翘 | 偶尔点名 | 每节签到 |
| 学分置换 | 大厂可抵 | 需审批 | 不允许 |
| 请假难度 | 秒批 | 能批 | 不放人 |
| GPA难度 | 宽松 | 正常 | 压分严重 |

### 5. 实习拉力折扣 pullDiscount（0~0.35）

**好城市 + 好区位 = 实习机会多 = 学校的边际价值下降**

```
geoAdvantage = city × 0.6 + location × 0.4
pullDiscount = (geoAdvantage / 10) × 0.35
```

- 城市/区位都满分 → 折扣35%
- 城市/区位都最低 → 折扣0%

**注意**：专业乘数和实习拉力是独立的：
- 专业乘数决定"学校资源对你有没有用"
- 实习拉力决定"外面的机会值不值得跑出去"
- 两者共同影响最终分数
